Movable Mobile Menu is a menu which appeares on mobile devices for improved navigation.

== Description ==

The Movable Mobile Menu provides a floating button on mobile devices which can be moved to user needs. With growing screens it is difficult to reach the navigation drawer especially when trying to work with one hand. The Movable Mobile Menu can be dragged to a comfortable position on the screen with little disturbance of the website.

== Installation ==

1. Upload 'movablemobilemenu.zip' through Wordpress 'Plugins' installation menu
2. Activate the plugin through the 'Plugins' menu in WordPress

== Frequently Asked Questions ==

= Why having a Movable Mobile Menu? =

In times when more and more use a mobile device to browse the internet, 
give your visitors the opportunity to decide where they prefer to have the navigation drawer.

= How does it work? =

The Movable Mobile Menu has two areas: one for dragging it and one for open a modal which contains your menu.
The menu appears on the top position of the Movable Mobile Menu, so it is easy reachable.
Tapping the modal outside the menu closes the modal and menu.

= Does it work with any menu? =

After installing and activating you got a new menu in Appearance -> Menus -> Display location named "Movable Mobile Menu".
You have to create a menu that you connect to this menu-position.
This one then is used as your Movable Mobile Menu.

= How much space need the Movable Mobile Menu? =

It adds a 50px wide and 120px high button to your site.
This button has 60% transparency which prevents to disturb your website.
